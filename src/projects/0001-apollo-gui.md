This is my first attempt at a resource pack for Minecraft. A Dark GUI, meant to feel modern and easy on the eyes, as well as provide utilities like a potion making guide.

It uses the pallete <a href="https://lospec.com/palette-list/apollo">Apollo</a> created by <a href="https://www.youtube.com/adamcyounis">AdamCYounis</a>, an incredibly talented game developer and pixel artist that has taught me a lot about pixel art.
